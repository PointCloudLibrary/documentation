Detailed Description

template<class MeshT>
class pcl::geometry::OuterHalfEdgeAroundFaceCirculator< MeshT >

Circulates clockwise around a face and returns an index to the outer half-edge (the target).

The best way to declare the circulator is to use the method pcl::geometry::MeshBase::getOuterHalfEdgeAroundFaceCirculator ().

Template Parameters
MeshTMesh to which this circulator belongs to.
Note
The circulator can't be used to change the connectivity in the mesh (only const circulators are valid).
