Detailed Description

template<typename PointT>
class pcl::visualization::PointCloudColorHandlerLabelField< PointT >

Label field handler class for colors.

Paints the points according to their labels, assigning a unique color from a predefined color lookup table to each label.

Sergey Alexandrov

Obtain the actual color for the input dataset as a VTK data array.

Deriving handlers should override this method. The default implementation is provided only for backwards compatibility with handlers that were written before PCL 1.10.0 and will be removed in future.
