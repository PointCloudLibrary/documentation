124 /** \brief KdTreeFLANN is a generic type of 3D spatial locator using kD-tree structures.

128  * \author Radu B. Rusu, Marius Muja

217  /** \brief Search for k-nearest neighbors for the given query point.

238  /** \brief Search for all the nearest neighbors of the query point in a given radius.

246  * \param[out] k_sqr_distances the resultant squared distances to the neighboring
