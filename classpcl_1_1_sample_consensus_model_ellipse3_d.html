Detailed Description

template<typename PointT>
class pcl::SampleConsensusModelEllipse3D< PointT >

SampleConsensusModelEllipse3D defines a model for 3D ellipse segmentation.

The model coefficients are defined as:

  • center.x : the X coordinate of the ellipse's center
  • center.y : the Y coordinate of the ellipse's center
  • center.z : the Z coordinate of the ellipse's center
  • semi_axis.u : semi-major axis length along the local u-axis of the ellipse
  • semi_axis.v : semi-minor axis length along the local v-axis of the ellipse
  • normal.x : the X coordinate of the normal's direction
  • normal.y : the Y coordinate of the normal's direction
  • normal.z : the Z coordinate of the normal's direction
  • u.x : the X coordinate of the local u-axis of the ellipse
  • u.y : the Y coordinate of the local u-axis of the ellipse
  • u.z : the Z coordinate of the local u-axis of the ellipse

For more details please refer to the following manuscript: "Semi-autonomous Prosthesis Control Using Minimal Depth Information and Vibrotactile Feedback", Miguel N. Castro & Strahinja Dosen. IEEE Transactions on Human-Machine Systems [under review]. arXiv:2210.00541. (@ github.com/mnobrecastro/pcl-ellipse-fitting)

Author
